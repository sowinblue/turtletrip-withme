# 거북이의 체크리스트 (가제)
‘거북이의 체크리스트’는 거북이와 함께 천천히 꾸준하게, 지루한 일상을 재미있는 성취의 여정으로 바꾸는 **웹 사이트** 기반 체크리스트 서비스입니다.

### 주요 기능
1. 할 일을 손쉽게 계획하고 수정 할 수 있습니다!
2. 목표 달성 시 거북이가 섬에서 바다를 향해 나아갑니다!
3. 달성한 목표는 바다 속으로 뽀글뽀글 가라앉습니다!
4. 상태바를 통해 달성률을 한 눈에 알아볼 수 있습니다!

### 기술 스택
- **언어**: Python
- **데이터 관리**: JSON
- **실행 환경**: 로컬 포트
- **핵심 기능**
    - 체크리스트 CRUD
    - 상태바(progress) 기반 달성률 계산
    - 거북이 캐릭터 메인 사용
